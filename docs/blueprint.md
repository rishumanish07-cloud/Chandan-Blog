# **App Name**: FireBlog

## Core Features:

- Blog Post Creation: Users can create and submit blog posts with text and images.
- Image Upload and Storage: Store uploaded images in Firebase Storage and save the file path in Firestore.
- Real-time Updates: Use Firestore to enable real-time updates of blog posts.
- Like/Unlike Functionality: Users can like or unlike blog posts, with the number of likes stored in Firestore.
- Commenting System: Users can comment on blog posts; comments are stored in Firestore.
- AI-Powered Post Summarization: Tool for using AI to generate a short summary of each blog post.
- User Authentication: Secure user authentication to manage and create content using credential login and Google Auth provider.
- AI-Powered Content Generation: Tool for generating blog post paragraph releted to title using AI

## Style Guidelines:

- Primary color: Deep indigo (#3F51B5) for a modern and professional feel.
- Background color: Light grey (#F5F5F5) for a clean and readable layout.
- Accent color: Teal (#009688) for interactive elements and highlights.
- Body font: 'Inter', a grotesque-style sans-serif for readability.
- Headline font: 'Space Grotesk', a proportional sans-serif for headers. If longer text is anticipated, use 'Inter' for body
- Use a consistent set of icons for navigation, likes, and comments, sourced from a library like Material Design Icons.
- Design a responsive layout that adapts to different screen sizes and devices.
- Incorporate subtle animations for loading indicators and transitions between pages.
